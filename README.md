# Spotify-API-Music-Data-Analysis
An exploratory analysis of artist performance using the Spotify API. This project demonstrates how to retrieve, process, and analyze music metrics to uncover trends in song popularity and listener engagement.

Key Insights
API Integration: Successfully established a connection with the Spotify API to extract real-time metadata, including track popularity, danceability, and audio features.

Performance Analysis: Identified top-performing tracks and artists by analyzing streaming metrics and spikes in popularity.

Trend Visualization: Visualized the relationship between song attributes and market success, identifying which musical characteristics drive listener engagement.

Technologies Used
Languages: Python

APIs: Spotify Web API (Spotipy library)

Libraries: Pandas, Matplotlib, Seaborn

Methodology
Data Extraction: Authenticated with the Spotify API to pull track data for specific artists.

Data Processing: Cleaned the raw JSON/data output to filter and organize metrics such as popularity scores and track duration.

Exploratory Data Analysis (EDA): Performed statistical analysis to compare track performance and identify patterns in listener preferences.

How to explore
Open the spotify_analysis.ipynb file directly in your browser.

If you want to experiment with the API yourself, ensure you have your own client_id and client_secret from the Spotify Developer Dashboard.
